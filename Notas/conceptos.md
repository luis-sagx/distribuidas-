# Aplicaciones Distribuidas

## 1. Programa

Un programa es la unidad más básica. Es un conjunto de instrucciones escritas en un lenguaje de programación que una computadora ejecuta para realizar una tarea específica.

Naturaleza: Es un archivo ejecutable o un script (ej. un archivo .exe o .py).

Estado: Puede ser algo tan simple como una calculadora de terminal o un script que organiza archivos. No necesariamente tiene una interfaz gráfica.

Ejemplo: Un script de Python para renombrar imágenes o el comando ls en Linux.

## 2. Aplicación (Software de Aplicación)

Una aplicación es un tipo de programa (o conjunto de programas) diseñado específicamente para que el usuario final realice una actividad coordinada.

Naturaleza: Está enfocada en la interacción humana y la utilidad (productividad, entretenimiento, comunicación).

Requisito: Siempre corre sobre un sistema operativo.

Ejemplo: Microsoft Word, WhatsApp, Spotify o Photoshop.

## 3. Sistema (Sistema Informático / de Software)

Un sistema es el concepto más amplio. Es una colección de componentes interconectados (hardware, software, redes y, a veces, personas) que trabajan juntos para lograr un objetivo complejo.

Naturaleza: Es una infraestructura completa. Un sistema puede contener múltiples programas y aplicaciones.

Alcance: Incluye desde el sistema operativo hasta la base de datos y los servidores.

Ejemplo: Un Sistema de Gestión Hospitalaria (que incluye la app del médico, la base de datos de pacientes y los servidores de red).
