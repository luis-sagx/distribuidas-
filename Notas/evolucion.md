# Evolución de los Sistemas Informáticos

## Decada 60-70: Origenes y Primeros Sistemas

### 1960:

- primeros sistemas operativos multiusuarios y multitarea como CTSS (MIT, 1961) y Unix (AT&T Bell Labs, 1969)
- necesidad de compartir recursos entre usuario remotos

### 1969:

- ARPANET, precursor de Internet, permite la interconexión de computadoras a través de redes
- punto de partida de la computacion distribuida

### 1972:

- correo electronico

## Decada 1980: Redes Locales y Cliente-Servidor

### 1880:

- surge la LAN como Ethernet
- modelo cliente-servidor como estandar para aplicaciones distribuidas

### 1882:

- conferencia Symposium on Principles of Distributed Computing (PODC) establece fundamentos teoricos

### 1985:

- proyecto ANSA (Advanced Network Systems Architecture) de DARPA impulsa la investigacion en sistemas distribuidos

### 1989:

- se funda el Object Management Group (OMG) que mas tarde desarrolla CORBA, un estandar para sistemas distribuidos orientados a objetos

## Decada 1990: Onjetivos Distribuidos y Middleware

### 1990:

- primera especificacion de CORBA (Common Object Request Broker Architecture) para facilitar la comunicacion entre objetos distribuidos

### 1992:

- aparece las primeras implementaciones comerciales de CORBA

### 1993:

- se introduce CGI (Common Gateway Interface) para permitir la creacion de paginas web y la interaccion entre servidores web y aplicaciones distribuidas

### 1995:

- java introduce RMI (Remote Method Invocation) permitiendo la invocacion de metodos entre objetos java en diferentes maquinas

### 1999:

- aplicaciones cliente-servidor se vuelven comunes en empresas y organizaciones, con herramientas como DCOM (Microsoft) Y CORBA en diversas industrias

## Decada 2000: Web Services y SOA

### 2000:

- Roy Fielding introduce REST (Representational State Transfer) sentando las bases para la arquitectura de servicios web moderna
- web services es un enfoque para construir aplicaciones distribuidas utilizando protocolos web estandarizados como HTTP, XML y SOAP

### 2001-2003:

- surge SOAP (Simple Object Access Protocol) como un protocolo estandar para servicios web basados en XML

### 2005:

- arquitectura orientada a servicios (SOA) se vuelve popular para diseñar sistemas distribuidos flexibles y reutilizables

### 2007:

- primeras plataformas de computacion en la nube, como Google App Engine (2008) sentando las bases para el despliegue de aplicaciones distribuidas en la nube
